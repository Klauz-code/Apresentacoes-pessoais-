Função Linear

Disciplina: Matemática | Tema: Funções

O que é Função Linear?

Função linear é uma função do tipo y = ax + b, onde o valor de y varia de forma proporcional a x.

O gráfico é uma reta, mostrando crescimento ou decrescimento constante.

Elementos da Função

a → coeficiente angular (inclinação da reta)
b → coeficiente linear (onde corta o eixo y)
x → variável independente
y → resultado

Comportamento

- a > 0 → função crescente
- a < 0 → função decrescente
- a = 0 → função constante

Exemplo 1

Função: y = 2x + 1

Cálculo:

x = 0 → y = 1
x = 1 → y = 3
x = 2 → y = 5

Conclusão: a cada aumento de 1 em x, y aumenta 2.

Exemplo 2

Função: y = -x + 4

Cálculo:

x = 0 → y = 4
x = 2 → y = 2
x = 4 → y = 0

Conclusão: a função é decrescente.

Gráfico

O gráfico de uma função linear é sempre uma reta. Basta encontrar dois pontos e ligá-los.

Resumo

Forma geral: y = ax + b
Gráfico: reta
m: inclinação
b: ponto inicial
